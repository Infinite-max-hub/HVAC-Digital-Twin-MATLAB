# HVAC Digital Twin – MATLAB

## Project Overview

This project presents a MATLAB-based digital twin of an office HVAC room system designed to simulate indoor thermal behaviour, occupancy heat gains, weather impact, and smart control strategies for energy optimisation.

## Objectives

* Predict indoor room temperature over 24 hours
* Simulate Melbourne outdoor weather influence
* Model occupancy heat gains during work hours
* Compare standard thermostat vs smart predictive control
* Reduce HVAC energy use while maintaining comfort

## Tools Used

* MATLAB
* Engineering Heat Transfer Principles
* Control Systems Logic
* Data Visualisation

## Key Results

* Stable indoor comfort temperature (22–24°C)
* Dynamic HVAC response to weather and occupancy
* Daily energy consumption analysis
* Smart control reduced simulated energy demand
